# Enhanced-for-loop-in-2-D-array
Enhanced for loop completely scan the array. It is also known as 'for each' loop.
class Test14
{
 public static void main(String args[])
{
 int num[][]={{11,22},{33,44},{55,66}};
 for(int i[]:num)
{
  for(int j:i)
{
System.out.print(j+" ");
}
System.out.println();
}
}
}
